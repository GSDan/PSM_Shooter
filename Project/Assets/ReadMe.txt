This is an example with both asset AssetBundles and scene AssetBundles.

1. Import the package
2. Create the asset bundles by right-clicking the StreamingAssets folder.
2.1 Right click, and select 'Build all Prefab AssetBundles PSM'
2.2 Right click, and select 'Build export SceneBundle PSM"'
3. Select/load the TestScene
4. Build&Run as usual (you can also test it in the editor).
